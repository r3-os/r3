Utility

This module is exempt from the API stability guarantee unless specified otherwise. It’s exposed mostly because it’s needed by macros.

Re-exports

  • Zeroable, ZeroableInOption: Re-exported from bytemuck ^1. Zeroable is a marker trait used in HunkDefiner::zeroed and suchlike. It can be derived for struct types. These re-exports are subject to the application-side API stability guarantee.
